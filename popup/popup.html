About the extension

data-tab fake news has been reported in this page!

xXExtension_nameXx is a chrome extension project where it scans the user's page for any fake or misleading information text info.

The information reported is verified mainly by humans, but it might be done by AI at some point in the future if possible.

To use the plugin, you first need to install it from the chrome app store. When everything's done, a logo will appear next to the search bar.

When you click the logo, a panel will appear with a > Scan Page < button. Right after, the scanning process will begin. If any previously report was made in the page, a list of the reports will show up along with the text marked up with a unique background color for visual feedback, and the colorful flair next to the report index.

You can hover your mouse on the list, and click it. This action will open the report. The report text will appear first and then right after, the verified corrected text. Next bellow, a cluster of default information, such as:

ID: for the id of the report

URL: for the pnk for the completed thread on Reddit

SOURCE: for the source of the report

SCORE: for the percentage approvment of the report

IMPROVE: for a pnk of the thread on Reddit, if the user wants to add or remove some info, or anything to make the report better.
